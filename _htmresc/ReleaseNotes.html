STM32CubeWBA Firmware Package V1.2.0 / 02-November-2023

Main Changes

Official Release of STM32CubeWBA Firmware package supporting STM32WBA52xx and STM32WBA55xx devices

Connectivity

- Bluetooth Low Energy 5.4 with new features: Enhanced ATT, Isochronous Broadcaster / Synchronizer, Connected Isochronous, AOA/AOD, LE Power Control

- Bluetooth Low Energy Audio: Generic Audio Framework Stack (CAP, BAP, VCP, MICP, MCP, CCP, CSIP), Use Case Profile : PBP, LC3 Codec

- Zigbee stack Revision 23 (R23) / Zigbee Clusters ZCL 8.0

- OpenThread: compliant Thread 1.1, 1.2 and 1.3

- Mac 802.15.4: compliant Zigbee IEEE 802.15.4 MAC Test Plan V2

Drivers

- CMSIS Device Drivers supporting STM32WBA52xx and STM32WBA55xx devices

- HAL/LL Drivers available for all peripherals

- BSP Drivers supporting NUCLEO-WBA52CG, NUCLEO-WBA55CG and STM32WBA55G-DK1 boards

Middlewares

- MW ST OpenBL V6.1.0 support

- MW AzureRTOS v6.2.0 support

- MW SBSFU/TFM support

Projects

- Add Applications to demonstrate Bluetooth Low Energy capabilities with STM32WBA55x

- Add new Applications demonstrating Zigbee, Thread and MAC 802.15.4 capabilities

- Add new Applications demonstrating BLE Audio

- Add LL/HAL Examples demonstrating the same peripherals as NUCLEO-WBA52CG and add new COMP, PWR, RCC LL/HAL Examples on NUCLEO-WBA55CG

- Add LL/HAL Examples demonstrating main system and SAI peripherals on STM32WBA55G-DK1
