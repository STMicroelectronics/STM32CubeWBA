STM32CubeWBA Firmware Package V1.6.0 / 05-February-2025

Main Changes

Official Release of STM32CubeWBA Firmware package supporting STM32WBA50xx, STM32WBA52xx, STM32WBA54xx, STM32WBA5Mxx, STM32WBA55xx, STM32WBA62xx, STM32WBA63xx, STM32WBA64xx and STM32WBA65xx devices

Security

- Minor updates in security applications (SBSFU and TFM) and related scripts (NUCLEO-WBA55CG)

- For security Applications, main board is STM32WBA65I-DK1

- Introducing OEMiRoT & OEMiRoT_OEMuRoT

- Add support for initial installation of clear image binaries in execution slots for OEMxRoT

- ROT provisioning scripts for OEMiRoT & OEMiRoT_OEMuRoT

- Introducing MbedTLS_HW_ALT and MbedTLS_SW

Connectivity

- BLE compliancy to TCRL 2024-2

- BLE Audio:

- LE Audio Projects available on STM32WBA65I-DK

- New GMAP Peripheral and GMAP Central projects available on STM32WBA65I-DK

- Add Scan Delegator feature in TMAP Peripheral project

- RAM Usage optimization in Audio projects

- Support of CubeMX for the different concurrent modes: BLE_Thread, BLE_Zigbee and BLE_MAC

- ioc files are available for applications using those 3 types of concurrent modes.

- Zigbee Direct :

- Zigbee Direct is a new communication protocol designed to enable direct communication between Zigbee devices and smartphones.

- Simple Application example using ZigbeeDirect are available in the BLE_Zigbee folders on Nucleo Boards.

- LSI2 support for Connectivity applications

Drivers

- CMSIS Device Drivers supporting TM32WBA50xx, STM32WBA52xx, STM32WBA54xx, STM32WBA55xx, STM32WBA5Mxx, STM32WBA62xx, STM32WBA63xx, STM32WBA64xx and STM32WBA65xx devices

- HAL/LL Drivers available for all peripherals

- HAL: ADC, COMP, CORTEX, CRC, CRYP, DMA, EXTI, FLASH, GPIO, GTZC, HASH, HSEM, HCD, I2C, ICACHE, IRDA, IWDG, LPTIM, PKA, PWR, PCD, RAMCFG, RCC, RNG, RTC, SAI, SMARTCARD, SMBUS, SPI, TIM, TSC, UART, USB, USART, WWDG

- LL : ADC, COMP, CRC, DMA, EXTI, GPIO, I2C, ICACHE, LPTIM, LPUART, PKA, PWR, RCC, RNG, RTC, SPI, TIM, USART, UTILS

- Update HAL/LL drivers to include latest corrections and to support of STM32WBA6xx devices

- BSP Drivers : Update STM32WBAxx_Nucleo, STM32WBA65I-DK1 and STM32WBA55G-DK1 BSP drivers to include latest corrections.

- Add B-WBA5M-WPAN BSP drivers

Projects

- B-WBA5M-WPAN

- Update new applications demonstrating BLE, Thread and Zigbee capabilities

- NUCLEO-WBA55CG

- Update Applications demonstrating BLE, Thread and Zigbee capabilities

- Add new Application demonstrating ThreadX and FileX capabilities

- STM32WBA55G-DK1

- Update applications demonstrating BLE, Zigbee capabilities

- NUCLEO-WBA65RI

- Add new Applications demonstrating BLE, Thread and Zigbee capabilities

- Add new Application demonstrating ThreadX, FileX and USBX capabilities

- Add HAL and LL examples

- STM32WBA65I-DK1

- Add new Applications demonstrating BLE, Thread and Zigbee capabilities

- Add new Application demonstrating OEMiRoT capabilities

- Add new Applications demonstrating mbedtls capabilities (MbedTLS_HW_ALT and MbedTLS_SW)

- Add a HAL SAI example for audio codec
