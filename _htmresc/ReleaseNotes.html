STM32CubeWBA Firmware Package V1.9.0 / 04-February-2026

Main Changes

Release Candidate of STM32CubeWBA Firmware package supporting STM32WBA20xx, STM32WBA23xx, STM32WBA25xx, STM32WBA50xx, STM32WBA52xx, STM32WBA54xx, STM32WBA5Mxx, STM32WBA55xx, STM32WBA62xx, STM32WBA63xx, STM32WBA64xx and STM32WBA65xx devices

- Introducing following new board support: NUCLEO-WBA25CE.

Security

- Secure boot:

- RTC/Tamper management improvements in OEMiRoT

- Improve python V3 installation dependency

- Ymodem protocol update for Linux compatibility

Connectivity

- BLE:

- Qualified against Bluetooth® Core 6.2

- The following features are now supported on the WBA25: Periodic Advertising with Responses (PAwR), LE Frame Space Update, Monitoring Advertisers, LL Extended Feature Set

- HCI event architecture has been reworked for Flash memory footprint improvement

- BLE Audio:

- No update (moved to BLE section)

- 802.15.4 General:

- LSI2 support has been added.

- Zigbee:

- Default libraries is now with **no traces**. Added new Libraries with traces.

- Thread:

- No update

Drivers

- CMSIS Device Drivers supporting STM32WBA20xx, STM32WBA23xx, STM32WBA25xx, STM32WBA50xx, STM32WBA52xx, STM32WBA54xx, STM32WBA55xx, STM32WBA5Mxx, STM32WBA62xx, STM32WBA63xx, STM32WBA64xx and STM32WBA65xx devices

- HAL/LL Drivers available for all peripherals

- HAL: HAL: ADC, COMP, CORTEX, CRC, CRYP, DMA, EXTI, FLASH, GPIO, GTZC, HASH, HCD, HSEM, I2C, ICACHE, IRDA, IWDG, LPTIM, OTFDEC, PCD, PKA, PWR, RAMCFG, RCC, RNG, RTC, SAI, SMARTCARD, SMBUS, SPI, TIM, TSC, UART, USB, USART, WWDG, XSPI

- LL : ADC, BUS, COMP, CORTEX, CRC, DLYB, DMA, EXTI, GPIO, HSEM, I2C, ICACHE, IWDG, LPTIM, LPUART, PKA, PWR, RCC, RNG, RTC, SPI, SYSTEM, TIM, USART, USB, UTILS, WWDG

- Update HAL/LL drivers to include latest corrections and to support of STM32WBA2xx devices

- BSP Drivers : Update STM32WBAxx_Nucleo to include latest corrections and to support of STM32WBA2xx devices

Projects

- Update Applications, Examples and Templates for following boards:

- B-WBA5M-WPAN

- NUCLEO-WBA55CG

- STM32WBA55G-DK1

- B-WBA6M-WPAN

- NUCLEO-WBA65RI

- STM32WBA65I-DK1

- Add support of new NUCLEO-WBA25CE board including Applications demonstrating BLE, Thread, Zigbee, RoT & Crypto capabilities, as well as Examples and Templates.
